## S3

- 이미지 올려보기
  - Amazon S3 > Bucket > Create bucket
    - Bucket name : luckybucket1
    - Block al public acess : 외부에서 접근 못하게 하기 때문에 용도에 따라 체크 해제
    - 경고메시지가 나옴 : 체크
    - Create bucket 버튼
  - 테스트
    - 버킷 선택 > Upload
      - add files : 이미지 업로드
    - 업로드 후 열어보기
  - 버킷의 권한과 오브젝트의 권한은 다름
    - luckybucket1 > Edit bucket policy > Policy Generator
      - Select Type of policy : S3 Bucket Policy
      - Effect : Allow
      - Principal : *
      - Action : getObject 체크
      - Amazon Resource Name (ARN) : [Bucket의 ARN 붙여넣기] + [/*]
      - Add Statement 버튼
      - Generate Policy 버튼
      - Json 복사
      - Bucket Policy에 붙여넣기
      - Save changes 버튼
- 정적 웹사이트 만들어보기
  - Amazon S3 > Bucket > luckybucket1
  - Properties탭
    - Static website hosting > Edit
      - Static website hosting : Enable
      - Index document : index.html
      - Error document : error.html
