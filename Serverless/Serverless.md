## Serverless란

- 서버를 우리가 관리할 필요 없이 알아서 우리들의 어플리케이션 코드를 클라우드에서 돌릴 수 있게 해줌
- AWS는 Infrastructrue Management를 알아서 해줌, 우리가 할일은 코드를 짜고 구현만 하면 됨

## AWS는 우리를 위해서 무엇을?
- Capacity Provisioning
- Auto-Scaling
- Patching
  - 리소스를 버전을 알아서 업데이트를 해주는 개념

## Serverless 이점
- 빠른 배포 (Faster Deployment)
- 대단한 확장성 (Scalable)
- 비용 절감 (Lower Cost)
  - 항상실행 되는것이 아닌, 이벤트에 의한 실행

## 대표적인 Serverless 서비스
- Lambda
- SQS
- SNS
- API Gateway
- DynamoDB
- S3
